ЮПИ! - CMS на Yii

Юпи! — мини CMS, написанная на замечательном MVC-фреймворке Yii (http://www.yiiframework.com/), на которой работает уже более 150 проектов.

Требования к хостингу

  • PHP >= 5.3
  • Yiiframework >= 1.1.13
  • MySQL/PostgreSQL
  • Apaсhe/Nginx

Возможности

Из коробки Вы получаете каркас приложения, со следующим уже готовым функционалом (всё разделено на модули - используйте только то, что необходимо):

  • Регистрация, аутентификация, восстановление пароля (модуль user)
  • Модуль пользователей, для управление пользователями сайта (блокировка, активация, редактирование и т.д.) через административный интерфейс (модуль user)
  • Модуль новостей для создания и публикация новостей на сайте (модуль news)
  • Модуль статических страниц, для создания и управление статическими страницами сайта (модуль page)
  • Модуль категорий, для создания и управление категориями вашего сайта (разделами) (модуль category)
  • Модуль меню, для создания и редактирование меню вашего сайта (модуль menu)
  • Модуль комментариев (можно комментировать любую сущность с выстраиванием древовидных комментариев, пример http://yupe.ru/post/logotip-dlya-yupi.html (модуль comment)
  • Модуль простых справочников (хранение и управление справочной информацией) (модуль dictionary)
  • Модуль для ведения блогов (как индивидуальных, так и коллективных) (модуль blog)
  • Модуль для "Обратной связи" + раздел FAQ (модуль feedback)
  • Модуль для работы с Wiki - работает через модуль yeeki
  • Модуль документации, который позволяет создавать документацию для вашего сайта (модуль docs)
  • Модуль изображений (модуль image)
  • Модуль галерея изображений, позволяющий организовать галереи изображений на вашем сайте (модуль gallery)
  • Модуль для создания и редактирования произвольных блоков контента (модуль contentblock)
  • Модуль для создания заданий (модуль queue)
  • Модуль управления почтовыми сообщениями (модуль mail)
  • Модуль для установки системы (модуль install)
  • Удобная админ панель в стиле Twitter Bootstrap (Выглядит вот так)
  • Возможность генерировать CRUD в стиле Twitter Bootstrap
